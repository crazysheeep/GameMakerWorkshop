Introduction

Introduction

GameMaker is a program that allows you to easily create games with drag and drop style commands, while also supporting more advanced programming. We will be using GameMaker to demonstrate some programming and game principles - and creating an awesome game of course!

Please note that this tutorial was created with GameMaker: Studio for Windows in mind. There may be slight differences if you are using GameMaker Lite for Mac or other versions of GameMaker.
