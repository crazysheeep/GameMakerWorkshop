Room Extension and Enemies

Room Extension

Let's make our room a bit bigger by increasing the boundaries and also add some scrolling!

To lengthen the size of the room:

  1. Go to Room Properties
  2. Click on the settings tab
  3. Increase the width to 1600

To get the camera to follow your main character:

  1. Go to Room Properties
  2. Click on the views tab
  3. Check Enable the use of Views
  4. Check Visible when room starts
  5. Scroll down to Object following and select obj_player from the drop down menu
    Let Hbor = 640, Vbor = 480, Hsp = -1, Vsp = -1

Enemies

Enemies are a lot like spikes. You don't want to touch them. However, enemies can also move around!

First we need to make a path for the enemy to move on. Let's start with an enemy that simple patrols left and right.

  1. Right click on the Paths folder and click on create path (or click on the swiggly arrow icon)
  2. Name it path_straight_line
  3. Click on the Indicate the room to show as background button (shown)
  4. Add two points, by clicking on the grid

Now we create an enemy, and give it this path.

  1. Create a new object, obj_enemy (You might want to make this have precise collision checking too)
  2. Add Event → Create
  3. Move tab → Set Path
    Let path = path_straight_line, speed = 8, at end = reverse, relative = relative
  4. Create a new event, collision with obj_player
  5. Copy and paste the actions from obj_spike
  6. Put the enemy in your room
