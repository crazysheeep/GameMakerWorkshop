Sprites

Sprites

A sprite is essentially an image. We'll need images for everything in the game. Let's start with the player.

To create a sprite:

  1. Right click on the sprites folder and click Create Sprite (or click the pacman icon)


    2. OR



  2. Name your sprite spr_player
    Note: Any name would be valid, but we will use the convention of having spr_ at the start of the name for sprites to make it easy to tell that this is a sprite.


  3. Click on Edit Sprite
  4. In the Sprite Editor, click on the New Page icon and press ok. (Leave the width and height as 32x32. It represents the size of the image in pixels).


  5. Double click on image 0 and draw your player
  6. Click on the Tick icon to save your sprite
  7. Click on the Tick icon again to close the Sprite Editor
  8. Press OK to save the Sprite Properties

Draw the following sprites:

  • Tile
  • Spike
  • Enemy
  • Flag
  • Icicle
  • Anything else you want to include in your game
