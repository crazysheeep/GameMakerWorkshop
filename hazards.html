Hazards

The Spike of Doom

Let's add some hazards. These are things your player doesn't want to touch. For example, a SPIKE OF DOOM.

To make our spike of doom:

  1. First, go back to spr_spike's properties
  2. Check "Precise collision checking"
    Note: This means that the player will only die if they touch the drawn part of the spike. Without this, the player would die if they touched any part of the 32*32 hitbox.
  3. Create an object called obj_sprite
  4. Go to obj_spike Properties
  5. Add event → Collision → obj_player
  6. Main 1 tab → Destroy the instance
    Check Applies to other
  7. Main 1 tab → Restart the current room

Falling off the edge

Right now, if the player falls of the map, nothing happens. Let's make it so the player dies if they fall off the map.

  1. Right click on obj_spike and click on duplicate
  2. Name this new object obj_spike_invis
  3. Uncheck Visible
  4. Press OK
  5. Go to your room's properties
  6. Put invisible spikes where necessary
