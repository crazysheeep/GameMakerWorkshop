Multiple Levels and Advanced Hazards

Multiple Levels

Let's add some extra levels by creating some new rooms and linking them together!

To creat a new room:

  1. Right click on the Rooms folder and click Create Room
  2. Make another level

To connect the end of the first level to the second level:

  1. Create obj_flag
  2. Go to obj_flag's Properties
  3. Add Event → Collision → obj_player
  4. Main 1 tab → Go to Next Room
  5. Open up room0 and add an obj_flag at the end of the level






There's a lot of blank space here because there's nothing else to add. It's that simple!

Suggestion: you could have a room called victory_screen




If you want to have a choice of rooms to go to next, you can specify which room you want the flag to go to, using:

&tab;Main 1 tab → Different Room

So something like flag1 goes to the forest and flag2 goes to the mountains.

Advanced Hazards

Hazards (and enemies) can be a lot more interesting using alarms.

Let's create an object obj_ice that will hang from the ceiling. Every two seconds it will spawn an obj_icicle that will fall until it smashes on the floor or kills the player.

First, let's set up obj_ice. We will set Alarm0 to trigger 60 steps after the level is loaded.

  1. Create obj_ice and obj_icicle
  2. Add Event → Create
  3. Main 2 tab → Set Alarm
    Let number of steps = 60, in alarm number = Alarm0
    Note: By default, the room performs 30 steps per second. 60 steps is 2 seconds.

Now, when Alarm0 is triggered, we want to create a falling icicle, and set Alarm0 to 60 again.

  1. Add Event → Alarm → Alarm0
  2. Main 1 tab → Create Moving
    Let object = obj_icicle, x = 0, y = 0, speed = 8, direction = 270, check relative
  3. Main 2 tab → Set Alarm
    Let number of steps = 60, in alarm number = Alarm0

Now let's set up obj_icicle. First, it should kill the player and restart the room when it hits the player.

  1. Go to obj_icicle's properties
  2. Add Event → Collision → obj_player
  3. Main 1 tab → Destroy instance
    Applies to other
  4. Main 1 tab → Restart room

Next, it should be destroyed when it hits the floor.

  1. Go to obj_icicle's properties
  2. Add Event → Collision → obj_tile
  3. Main 1 tab → Destroy instance
    Applies to self
